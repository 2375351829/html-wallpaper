# 壁纸世界

一个精美的壁纸分享网站，提供各类高清壁纸免费下载。

## 项目简介

壁纸世界是一个纯静态的壁纸分享网站，提供多种分类的高清壁纸下载，包括风景、动漫、电影、游戏、水墨画等。网站界面简洁美观，采用响应式设计，完美支持移动端和桌面端访问。

## 功能特性

### 核心功能
- **轮播图展示**：首页轮播图自动播放（5秒切换），支持手动切换和指示器导航
- **分类浏览**：6大分类（风景、动漫、电影、游戏、水墨画、其他），共39张壁纸
- **图片预览**：点击预览按钮查看大图，支持左右切换和键盘操作（方向键、ESC）
- **壁纸下载**：一键下载壁纸到本地
- **收藏功能**：收藏喜欢的壁纸（心形图标）
- **搜索功能**：搜索框 UI 已实现（功能开发中）

### 交互体验
- **返回顶部**：滚动超过300px显示返回顶部按钮
- **平滑滚动**：锚点链接平滑滚动效果
- **滚动动画**：壁纸卡片和分类卡片滚动进入视图时动画效果
- **Toast 提示**：操作反馈提示消息
- **移动端菜单**：响应式导航栏，移动端汉堡菜单

### 筛选与排序
- 排序方式：最新、最热、最多下载
- 显示数量：12、24、48、全部
- 视图切换：网格视图、列表视图

### 页面功能
- **首页**：轮播图 + 分类快捷入口 + 热门壁纸展示
- **分类页**：所有分类卡片展示，显示壁纸数量
- **关于页**：网站介绍、使用说明、常见问题、统计信息、联系方式
- **分类详情页**：面包屑导航 + 筛选工具栏 + 壁纸网格展示

## 技术栈

### 第三方库
- **Bootstrap 4.4.1**：UI 框架
- **jQuery 3.7.1**：JavaScript 库

- 显示壁纸信息（分辨率、大小、下载次数）

### 懒加载
- 使用 IntersectionObserver API
- 图片进入视口前200px开始加载
- 提升页面加载性能

### 滚动动画
- 使用 IntersectionObserver API
- 元素进入视口时触发动画
- 提升用户体验

## 使用方法

### 方式一：直接打开
1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件

### 方式二：本地服务器（推荐）
1. 克隆本项目
2. 启动本地服务器
3. 在浏览器中访问对应地址

#### Python 3
```bash
python -m http.server 8000
```
访问：http://localhost:8000

#### Python 2
```bash
python -m SimpleHTTPServer 8000
```
访问：http://localhost:8000

#### Node.js (http-server)
```bash
npx http-server
```
访问：http://localhost:8000

#### VS Code Live Server
1. 安装 Live Server 扩展
2. 右键 `index.html` 选择 "Open with Live Server"

## 常见问题

### Q: 如何添加新壁纸？
A: 将壁纸图片放入对应的 `image/分类/` 目录，然后在对应的 HTML 文件中添加壁纸卡片。

### Q: 如何修改主题色？
A: 在 `assets/css/style.css` 文件顶部的 `:root` 中修改 CSS 变量：
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
}
```

### Q: 如何部署到 GitHub Pages？
A:
1. 将项目推送到 GitHub 仓库
2. 进入仓库 Settings -> Pages
3. 选择 main 分支作为 Source
4. 保存后等待几分钟即可访问
